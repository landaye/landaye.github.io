. 你是怎么理解ES6中 Promise的？使用场景？

Promise是异步编程的一种解决方案,有三种状态

  1. pending（进行中）
  2. fulfilled（已成功）
  3. rejected（已失败）
    Promise对象是一个构造函数，生成Promise实例,实例存在以下方法：
  4. then()
  5. catch()
  6. finally()

当下个异步请求依赖上个请求结果时，Promise的链式操作就很友好。

. 异步解决方案

  1. 回调函数
  2. Promise 对象
  3. generator 函数
  4. async/await

. Vue3.0里为什么要用 Proxy API 替代 defineProperty API ？

为了解决defineProperty API

  1. 检测不到对象属性的添加和删除
  2. 数组API方法无法监听到
  3. 需要对每个属性进行遍历监听，如果嵌套对象，需要深层监听，造成性能问题
