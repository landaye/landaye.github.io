. 说说对 React 的理解？有哪些特性？

  1. 用于构建用户界面的 JavaScript 库,虚拟 DOM 来有效地操作 DOM，遵循从高阶组件到低阶组件的单向数据流.

  2. 特性

    1. JSX 语法
    2. 单向数据绑定
    3. 虚拟 DOM
    4. 声明式编程
    5. Component

. 说说 Real DOM 和 Virtual DOM 的区别？优缺点？

  1. Virtual Dom，本质上是以 JavaScript 对象形式存在的对 DOM 的描述
  2. 区别:
    虚拟 DOM 不会进行排版与重绘操作,而真实 DOM 会频繁重排与重绘
